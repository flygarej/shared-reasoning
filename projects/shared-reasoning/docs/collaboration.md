# Collaboration

This document explains the collaboration philosophy behind Shared
Reasoning.

The methodology emerged through repeated reverse-engineering and
research projects rather than by designing a prompt from first
principles.

Core ideas:

-   Separate accepted knowledge from historical reasoning.
-   Treat documentation as evidence rather than authority.
-   Use the smallest experiment capable of distinguishing competing
    models.
-   Maintain explicit working models until evidence justifies promotion.
-   Improve collaboration by refining abstractions rather than
    accumulating rules.

The protocol is intended to make human and AI participants converge on a
shared conceptual model instead of merely exchanging answers.
