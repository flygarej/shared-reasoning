In order to merge domain knowledge from one or more chats, run the "prompt-each.md" prompt from this folder in each chat. Save the domain-area.md and documentation-view.md from each.

Start a new chat, give it capsules/new-reasoning.md, collaboration.md, knowledge-representation.md and reasoning-state.md from projects/shared-reasoning/domain-llm. Then give it the markdowns extracted from the chats.
Finally, give it prompt-merge.md from this folder.

When done, you should have a compact file enabling your LLM access to basic domain knowledgde gained from earlier chats.
